## 📦 **Project Architecture**

<div align="center">

<img src="https://readme-typing-svg.herokuapp.com?font=Roboto&size=22&duration=2500&pause=1000&color=FF6B6B&center=true&vCenter=true&width=600&lines=🏗️+Turborepo+Monorepo+Structure" alt="Architecture Header" />

</div>

```bash
📁 Data-Avengers-LMS/
├── 🌐 apps/
│   ├── 📱 data-avengers-lms/    # Main LMS Application (Next.js)
│   ├── 📱 data-avengers-main/   # Landing Page Application
│   ├── 📖 docs/                 # Documentation Site
│   └── 🌐 web/                  # Additional Web App
├── 🔧 backend/
│   ├── ⚡ express-server/       # REST API Server
│   └── 🔌 websocket-server/     # Real-time Communication
├── 🗄️ database/
│   └── 📊 schemas/              # Prisma Database Schemas
├── 📚 packages/
│   ├── 🎨 shadcn/              # UI Component Library
│   │   └── 🎯 next/            # Next.js Optimized Components
│   ├── 🔄 redux-toolkit/       # State Management + Persistence
│   ├── 📝 common/              # Shared Zod Schemas & Types
│   ├── 🎨 tailwind/            # Tailwind CSS Configuration
│   ├── ⚙️ eslint-config/       # ESLint Rules & Standards
│   └── 📋 typescript-config/   # TypeScript Configurations
├── 📄 documents/
│   ├── 📋 COMMIT_GUIDELINES.md # Git Commit Standards
│   └── 🔧 ESLINT_HUSKY_GUIDE.md # Development Guidelines
└── 🚀 Configuration Files
    ├── 📦 package.json         # Root Package Configuration
    ├── 🔧 turbo.json          # Turborepo Pipeline Config
    ├── 🎯 eslint.config.js    # Global ESLint Rules
    └── 📋 tsconfig.json       # TypeScript Root Config
```
